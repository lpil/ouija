Ouija
=====

[![Build Status](https://travis-ci.org/lpil/ouija.svg?branch=master)](https://travis-ci.org/lpil/ouija)

All knowing boards.

## Setup

Make `config/dev.secret.exs`. See `config/dev.secret.exs.example`

```sh
# Install Elixir deps
mix deps.get
# Install Javascript deps
npm install

# Run the server
mix phoenix.server
```
